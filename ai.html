AI Muse

Unleash your creativity with Prompt AI! Analyze the given prompt and recreate or clone the specified design, code, or concept.

Description:

AI Muse is a creative AI-powered competition where participants must recreate a given image using an AI tool within a given time. The game tests players' memory, creativity, and prompt engineering skills. The challenge lies in observing the image, recalling details, and generating the most accurate recreation using AI.

Instructions:

  1. Each team is allowed to participate with one member.
  2. The game lasts for 1 hour.Participants are provided with an AI tool for image generation
  3. The original image is displayed for 3 minutes before creation begins
  4. After the observation phase, participants start recreating the image using prompts.
  5. Midway through the game, the image is shown again for 2 minutes for reference.
  6. Participants continue refining and finalizing their AI-generated images.
  7. All creations must be submitted before the 1-hour mark.
  8. Judges evaluate submissions based on accuracy, creativity, and prompt effectiveness.
  9. Winners are selected based on the highest scores given by the judges.
